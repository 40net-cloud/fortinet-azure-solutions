# Azure China - FortiGate Next-Generation Firewall - A Single VM

## Introduction

The deployment of the FortGate-VM in the Azure China region is similar to the Azure Global regions. More information about the global region deployment can be found [here](/FortiGate/A-Single-VM/README.md).

## Deployment

For the deployment, you can use the Azure Portal, Azure CLI, Powershell or ... The Azure ARM templates are exclusive to Microsoft Azure and can't be used in other cloud environments. The main template is the `azuredeploy.json` which you can use in the Azure Portal.
